### SwitchArray 2026-S000
+ 組裝日期: 2026-04-29
+ SwitchArray完整序號為: 2026-S000
+ 使用測試機殼。
+ 組裝Control_Board_v1.2
  + 取出一片Control_Board_v1.2，登記出廠序號「2025100001」。移除底部固定螺絲，螺絲另外收好。在H101附近貼上標籤貼紙「SwitchArray\n2026-S000」。
  + 先裝上BNC公轉杜邦母_30cm(型號BNC-MDP-03)連接線，紅色接V_ready，黑色接GND。
  + 再裝上ESP32-WROOM-32E(ESP32-Devkit v3.0)
    + SwitchArray內部使用的「ESP32」一律改用「CP2102」版本
      + 使用CP210x修改序號工具將序號修改為: 「2026-S000」
        + 插上裝置後(避免弄錯一次只插上一個CP210x裝置)，運行「CP210xSetIDs.exe」，找到中間有一個格子有標註「Serial Number」，填入序號「2026-S000」，在把前面的勾選框勾起來，最後按下最下方的「Program Device」，完成後關閉修改工具，並反覆插拔裝置與檢查序號是否正確。
      + 修改成功後，可在裝置管理員>連接埠(COM和LPT)>Silicon Labs CP210x USB to UART Bridge (COM3)>詳細資料> 裝置例項路徑，會看到「USB\VID_10C4&PID_EA60\2026-S000」。其中「2026-S000」就是SwitchArray完整序號。
    + 燒錄韌體「SwitchArray_ver2_9.ino」。
    + 以上工作都完成後，於ESP32上IC的金鐘罩貼上標籤貼紙，上方貼的標籤貼紙「CP2102」有加外框字體6pt。下方貼的是標籤貼紙「S007」有加外框字體10pt。用剪刀稍微修剪再貼上去。
  + 鎖上配合使用的電源線:
    + 牛角座編號1: 準備配合使用的黑色電源線(從8芯纜線中取出黑色)，長度33[cm]，單端壓接Y型端子(型號Y1.25-3S)並用絕緣膠帶纏繞。
    + 牛角座編號2: 準備配合使用的灰色電源線(從8芯纜線中取出灰色)，長度33[cm]，單端壓接Y型端子(型號Y1.25-3S)並用絕緣膠帶纏繞。
    + 牛角座編號3: 準備配合使用的綠色電源線(從8芯纜線中取出綠色)，長度33[cm]，單端壓接Y型端子(型號Y1.25-3S)並用絕緣膠帶纏繞。
    + 牛角座編號4: 準備配合使用的紅色電源線(從8芯纜線中取出紅色)，長度33[cm]，單端壓接Y型端子(型號Y1.25-3S)並用絕緣膠帶纏繞。
    + 牛角座編號5: 準備配合使用的橙色電源線(從8芯纜線中取出橙色)，長度33[cm]，單端壓接Y型端子(型號Y1.25-3S)並用絕緣膠帶纏繞。
