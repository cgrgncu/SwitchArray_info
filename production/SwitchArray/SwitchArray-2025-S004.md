+ 準備配件收納包(SC-TI06長方型加大防潑水硬殼收納包):
  + 配件包正反面貼上標籤貼紙「S004」於中間，正反兩面各一張。
  + 配件1:
    + 優先使用:
      + USB2.0數據線(透明藍)_1m_A公對A公，一條。中間貼上標籤貼紙「USB2.0\nAtoA 1M」，有加外框字體10pt。兩個USB頭平坦側貼上標籤貼紙「S004」，有加外框1字體6pt。用六號夾鏈袋裝好，放入收納包中。

    + 棄用(不明原因目前選擇廠牌的1.5M會影響放電):
      + USB2.0數據線(透明藍)_1.5m_A公對A公，一條。中間貼上標籤貼紙「USB2.0\nAtoA 1.5M」，有加外框字體10pt。兩個USB頭平坦側貼上標籤貼紙「S004」，有加外框字體6pt。用六號夾鏈袋裝好，放入收納包中。
  + 配件2:PSU連接線(橘、灰)_槍型接頭轉防水接頭，一組。製作方式:
    + 準備槍型接頭(或稱德式香蕉插頭)紅色、黑色，各一個。
    + 從8芯纜線中取出橘色、灰色電線各一條，每條裁切為62[cm]長。
    + 準備KWAY綠色防水接頭一個。平坦側貼上標籤貼紙「S004」，有加外框字體10pt。
    + 先將一端剝線約1[cm]，焊接到槍型接頭上，可用夾子輔助。利用另一個接頭撬開尾部。安裝回去的時候也要利用另一個接頭。紅色接頭配橘色線，黑色接頭配灰色線。
    + 兩條接頭帶線完成後，接到電供上(不要通電)，使用蘋果膠帶把兩條線束緊固定。保留約3[cm]用於安裝防水接頭。
    + 焊接電線到防水接頭上，記得按照順序先把零件套到線上，橘色線焊接到PIN#1，灰色線焊接到PIN#2。最後組裝起來，並使用電錶查驗通路與否。
    + 用五號夾鏈袋裝好，放入收納包中。
  + 配件3:DMM連接線(紅、黑)_槍型接頭轉電錶接頭，一組。製作方式:
    + 準備槍型接頭(或稱德式香蕉插頭)紅色、黑色，各一個。
    + 從雙頭香蕉接頭測試線(FC-TL2098)中取出黑、紅色各一條，每條120[cm]從中間裁切一半長度，裁完變成約60[cm]長。至此變成黑紅各兩條。
    + 取出裁切好的紅、黑各一條，尾端剝線1.5[cm]，焊接到槍型接頭上，可用夾子輔助。利用另一個接頭撬開尾部。安裝回去的時候也要利用另一個接頭。紅色接頭配紅色線，黑色接頭配黑色線。
    + 兩條接頭帶線完成後，接到電錶上(不要通電)，使用蘋果膠帶把兩條線束緊固定，並貼上標籤貼紙「S004」。兩邊末端保留約3[cm]。
    + 用六號夾鏈袋裝好，放入收納包中。
  + 配件4:BNC雙公射頻線_1M_RG316(BNC-316U-100)，一條。中間貼上標籤貼紙「S004」，用六號夾鏈袋裝好，放入收納包中。
